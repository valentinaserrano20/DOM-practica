# DOM-practica

• ¿Qué elemento del DOM estás seleccionando?
    estoy seleccionando el boton, el id msg que corresponde a el text area donde se ingresa el mensaje y una clase mensaje que corresponde al div donde se almacenara el mensaje que el usuario escribio 

• ¿Qué evento provoca el cambio en la página?
    el boton del formulario

• ¿Qué nuevo elemento se crea?
    se creo un nuevo elemento "p" dentro de un contenedor div(.mensaje) para mostrar los mensajes que fueron agregados por el usuario

• ¿Dónde se inserta ese elemento dentro del DOM?
    en un nuevo elemento p que fue creado.

• ¿Qué ocurre en la página cada vez que repites la acción?
    Se van agregando los textos debajo del mensaje anterior
